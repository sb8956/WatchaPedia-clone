#### ⚖ Commit Convention

| 머릿말   | 설명                               |
| -------- | ---------------------------------- |
| feat     | 기능 구현                          |
| setting  | 패키지 설치, 개발 설정             |
| refactor | 기능변화 없이 최적화, 코드 개선 등 |
| fix      | 버그 수정                          |
| style    | 스타일링, 변수명 수정              |
| docs     | README.md 작성,주석 추가           |

#### ⚖ Branch Convention

모든 feature는 dev에서 분기

| 머릿말  | 설명                        |
| ------- | --------------------------- |
| master  | 최종 결과                   |
| dev     | 기능 구현                   |
| feature | 기능 단위 구현              |
| fix     | master에서 발견된 버그 수정 |
